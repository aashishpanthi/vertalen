Trilingual web translator

Welcome to vertalen

Read and translate the open web between English, Nepali, and Tamang. Optional immersion mode weaves vocabulary into pages as you read so you can pick up Tamang or Nepali in context — powered by the TMT translation API.

  1. 1

    Add your TMT API key

    vertalen needs your TMT API token for live translation (selections, full page, reader). Use the token issued for your account or project.

    Immersion learning uses built-in vocabulary and does not require this key — you can browse with learn mode first, then add your token when you are ready to translate.

  2. 2

    Try a sample translation

    Your translation will appear here.
  3. 3

    Learn the gestures

    🖱️

    Select text

    Highlight any phrase on a web page and click the floating Translate button.

    ⌨️

    Use shortcuts

    Alt + T translates a selection. Alt + Shift + T translates the whole page.

    📰

    Reader mode

    Open the popup and choose Open in side-by-side reader for long articles.

  4. 4

    Want to learn while you browse?

    Immersion mode sprinkles Tamang or Nepali vocabulary into the English pages you read. Hover any translated word to see the original, click for actions, and quiz yourself in the popup. It's an opt-in way to support language preservation just by browsing the way you already do.

    You can change this any time from the popup or Settings.

  5. 5

    You're all set

    Pin the vertalen icon to your Chrome toolbar to keep it one click away. You can revisit any of these settings later from the Options page.
