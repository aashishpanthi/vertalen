API key

Your TMT team token is stored locally in your browser profile and is sent only to tmt.ilprl.ku.edu.np. It is required for live translation. Immersion / Learn works offline from bundled vocabulary without a key.

Default languages

Used when auto-detect is off, and as the seed for the popup.

Behavior

Blocked sites

Full-page translate, reader mode, immersion vocabulary, and the on-page selection tooltip are skipped on these hosts so search results, social feeds, and bot-check pages stay readable. Google Search (/search on any google.* host) is always treated as a search page. Your popup translator is not affected.

Immersion / learn mode

Sprinkle Tamang or Nepali vocabulary into the English pages you read, hover to see the original, and quiz yourself in the popup. Designed to help everyday speakers contribute to keeping Tamang alive.

About

vertalen is an open-source Chrome extension for translating pages between English, Nepali, and Tamang, with an optional learn-on-the-go mode.

  • License: MIT
  • API endpoint: https://tmt.ilprl.ku.edu.np/lang-translate
